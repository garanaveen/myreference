@howto create a simple 9-patch image?
This is an example of a button.
First draw a square (say 20x20 pxl). If needed add ronded corners (say 3 pxl)
Apply a 9-patch plugin (file->plugin->9patch)
Now adjust the black line at the left and top appropriately set the stretchable area.
Also adjust the black line on the right and bottom to set the optional-content area.

@howto download an asset from @figma?
Select the icon,
On the right hand panel you will see "Export <>" button.
Pressing that will download the asset in a zip file.
